Place your logo here to brand the app and installers.

- Replace `icon.png` with a square 1024x1024 PNG (transparent background recommended).
- You can also add platform-specific versions:
  - `icon.ico` for Windows (optional; `icon.png` will be converted if missing).
  - `icon.icns` for macOS (optional; `icon.png` will be converted if missing).
- Keep filenames the same so build scripts pick them up automatically.
